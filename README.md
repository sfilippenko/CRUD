# CRUD

Мини-приложение для просмотра, добавления, редактирования и удаления задач

Стартовать только локально через ```webpack-dev-server``` командой

```
npm run dev
```

## Технологии

* React
* React-router
* Redux
* Redux-saga
* Formik
* Material-ui
* Typescript
* Webpack

### Для удобства разработки были использованы

* Eslint
* Prettier
* Husky
